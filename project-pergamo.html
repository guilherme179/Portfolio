Pergamo

Um sistema criado para o setor de RH, permitindo que gestores avaliem suas equipes por meio de formulários personalizados. O Pergamo é uma ferramenta essencial para identificar áreas de melhoria, orientar treinamentos e promover o desenvolvimento contínuo dos colaboradores, fortalecendo a eficiência organizacional.

Visão Geral do Projeto

O Pergamo é um sistema desenvolvido para o setor de RH, com o objetivo de otimizar e automatizar o processo de criação e aplicação de formulários para avaliação de equipes. Anteriormente, o RH utilizava o Google Forms, onde era necessário inserir manualmente a hierarquia de supervisores, tornando o processo trabalhoso e demorado. Com o Pergamo, o sistema busca automaticamente a hierarquia dos supervisores, eliminando esse esforço manual.

Além disso, o Pergamo oferece personalização de regras de negócio, uma funcionalidade que não era possível no Google Forms. O sistema foi projetado para ajudar o RH a identificar pontos de melhoria, destacar pontos fortes e otimizar os treinamentos, promovendo a constante evolução operacional da Tel.

O backend do projeto foi implementado com NestJS e comunicação REST, utilizando um banco de dados relacional para armazenar informações de avaliações. O sistema também conta com um serviço de envio de e-mails automatizados, alertando supervisores que ainda precisam avaliar suas equipes. Outra funcionalidade chave é o cálculo de aderência, que permite ao RH monitorar a taxa de avaliações realizadas em diferentes sites e cobrar ações para garantir que supervisores concluam suas tarefas.
