Mitra Reorientações

Um sistema desenvolvido para gerenciar reorientações dos operadores na Tel Contact Center. O Mitra permite que o setor de qualidade envie feedbacks detalhados após monitorias de atendimentos diretamente para os supervisores, garantindo que as orientações sejam aplicadas de forma eficaz, promovendo melhorias contínuas no desempenho dos operadores.

Visão Geral do Projeto

O Mitra Reorientações é um sistema desenvolvido para gerenciar reorientações de operadores na Tel Contact Center, com o objetivo de aumentar a eficiência operacional da empresa. O sistema permite que o setor de qualidade capte informações detalhadas de cada caso, avalie a tratativa do operador e identifique os pontos de melhoria. Com base nisso, são gerados feedbacks assertivos, permitindo que os operadores sejam corrigidos e orientados para aprimorar sua performance.

O processo envolve o supervisor como parte essencial, responsável por aplicar os feedbacks diretamente aos operadores. Além disso, o sistema conta com um serviço próprio para envio de e-mails automatizados, garantindo que todas as partes envolvidas — setores e colaboradores — sejam informadas sobre atualizações relacionadas a cada caso, promovendo transparência e eficiência no acompanhamento.

O Mitra foi desenvolvido utilizando tecnologias como PHP, HTML, CSS e JavaScript, com o backend implementado no Slim Framework. Essa combinação oferece leveza e flexibilidade para o gerenciamento de reorientações, tornando o sistema robusto e eficaz para atender às necessidades da Tel.
