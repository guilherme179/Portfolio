Suporte Mis

Um sistema de gerenciamento de chamados desenvolvido para o setor MIS da Tel, permitindo que colaboradores de outros departamentos enviem solicitações de suporte relacionadas à geração de relatórios, ajustes em sistemas e outras demandas específicas. O Suporte MIS centraliza as solicitações, facilitando o acompanhamento e garantindo maior eficiência no atendimento.

Visão Geral do Projeto

O Suporte MIS é um sistema desenvolvido para centralizar e facilitar os pedidos de suporte dentro da Tel Contact Center, tanto para respaldar os colaboradores que fazem as solicitações quanto para otimizar o trabalho do setor MIS, responsável por atender essas demandas. O sistema oferece um quadro de avisos para comunicar problemas com relatórios ou sistemas, proporcionando uma visão clara e organizada das pendências.

Desenvolvido com PHP e tecnologias front-end como HTML, CSS e JavaScript, o sistema utiliza o Slim Framework no backend para garantir leveza e eficiência. Ele conta com um banco de dados relacional para armazenar e gerenciar as solicitações, além de uma automação em Python que fecha automaticamente chamados pendentes de resposta por parte do solicitante após 60 dias, garantindo um fluxo mais ágil e organizado.

Com o Suporte MIS, as solicitações são centralizadas, os prazos são gerenciados e tanto os colaboradores quanto o setor MIS têm maior transparência e controle sobre as demandas, contribuindo para um ambiente de trabalho mais eficiente.
